# 太上老君灵签

一个静态的太上老君灵签抽签网站，包含签筒抽签动画、36 支原创签文、分主题解签、最近签文记录和复制签文功能。
自用为主，如果你觉得喜欢也欢迎免费使用。

## 预览

直接打开 `index.html` 即可使用。也可以用任意静态服务器托管本目录。

## 功能

- 36 支原创灵签签文
- 总览、事业、财运、姻缘、健康、学业分主题解读
- 抽签动画与结果签纸
- 最近签文本地记录
- 一键复制签文
- 桌面与手机自适应布局

## 说明

签文参考传统灵签的结构、意象和语气进行原创创作，没有照搬特定庙宇签谱或古籍原文。内容仅供静心参考和娱乐体验，不作为现实决策依据。
